Anberlin

Anberlin - американская группа, которая играет в стиле альтернативный рок, из города Винтер Хэвен во Флориде. Группа образована в 2002 году и с начала 2007 состоит из вокалиста Стивена Кристиана, бас-гитариста Деона Рексроута, соло-гитариста Джозефа Миллигана, барабанщика Натана Янга и ритм-гитариста Кристиана МакАлэни.
